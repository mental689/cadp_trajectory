Introduction

Introduction

To address the problem of predicting future trajectories of person and vehicles in surveillance applications. we collected and annotated a challenging set of traffic camera footages. This dataset is named Car Accident Trajectories (CAT) dataset, contains 2184 person/vehicle annotations from 205 videos which has been downloaded from YouTube. We divided it (by random sampling) to 1208 trajectories for training and 976 trajectories for testing. Details to download this CAT dataset can be found in Download section.

This CAT dataset was built for the following problem:
Problem: Given a trajectory of \(T+\Delta\) points \(\mathbf{s}_t,~t=1,\ldots,T+\Delta\) where \(\mathbf{s}_t=[x_t,y_t]\) is the coordinates of points in a 2D plane. We let the algorithm observe first \(T\) points, and we ask it to predict the next \(\Delta\) points. In another word, we want to build a computer algorithm \(f\) which can infer the future from past observations: $$f(\{\mathbf{s}_t\}_{t=1}^T)=\{\mathbf{s}_t\}_{t=T+1}^{T+\Delta}$$ A common setting for CAT dataset is \((T,\Delta)=(8,12)\), i. e., given 8 points from the past, predict next 12 points of the trajectory.
